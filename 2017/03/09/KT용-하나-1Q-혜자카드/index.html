KT용 하나 1Q 혜자카드

인터넷을 돌아다니다가 하나멤버스 1Q 리빙카드 라는 혜자카드를 발견

2017년 1월2일부터 6월30일까지 한시적으로 발급할수 있는 카드인데

카드 기능이 무려 1원이상 실적시 모든 KT 통신비 5천원 할인

( 인터넷 / 집전화 / 핸드폰 / 와이브로 까지.. )

거기다 KT 만이 아니라 SK 계열까지!! ( LG는 출시예정 )

우리카드도 질세라 비슷한 이벤트를 하고는 있는데 지원되는 통신사가 적었습니다.

- 우리 위비카드 현황 -

M모바일 롯데슈프림DC카드 https://www.ktmmobile.com/event/eventDetail.do?ntcartSeq=413&pageNo=1&sbstCtg=E&searchValue= (월 1건 이용시 12개월 1만원 할인 동일)

위비 제휴 통신카드는 중복발급이 불가능하므로 1장만 선택해서 발급

여기서 KT 홈페이지에서는 전월실적 제외에 상품권, 온라인쇼핑 전체 사용금액을 고지하고 있으나

2017년 1월 1일 변경된 하나카드 홈페이지를 보면 ( https://m.hanacard.co.kr/MOPNWS110M.web?AN_NO=3043)

1월1일부터 전원실적에 온라온 쇼핑 사용금액이 포함되는것을 알수 있다

이제 이 카드를 자세히 알아봅니다

더불어 이 카드는 출시 이벤트로 다음과 같은 이벤트를 진행하고 있었음

이동통신 자동이체 신규...가 아니니 해당이 안될듯하고 이벤트 번호 1번

1만 하나머니는 첫 이용금액 자체가 1만원이 넘는 통신비를 결제할테니 통과할것이다

와이브로 랑 핸드폰이랑 연결해 놓으면 대박일듯

신규 발급 시 수령 월 포함 2달간 전월 실적 무관하게 청구할인 혜택 제공

이벤트 혜택 비고 하나머니 적립 최초 카드 사용등록 후 첫 이용금액 1만 원 이상 시 결제계좌에 따라

최대 1만 하나머니 적립

- KEB 하나은행 결제계좌 시 : 1만 하나머니 적립

- 타행 결제계좌 시 : 5천 하나머니 적립 최초 1회 한정 SK주유소 주유 쿠폰 제공 SK주유소 건당 5만원 이상 결제 시 5천원의 주유 쿠폰을 제공 월 1회, 3개월간 1만원 캐시백 이동통신 자동이체 신규 신청 후 납부 시 1만원 캐시백을 제공 최초 1회 한정

하나 알아봐야 할게 현재 지금 올인원 자동납부 이벤트에 참가중이다.

기존의 내역을 보게 되면 8~10월달이 5천하나머니에 해당되어 지급중이고

4~12월달 즉 올해 8월달까지는 해달 2천 하나머니를 지급 받을거라 예상중

포인트 내역에 안보니길래 하나멤버스 내역에 보니 다음과 같이 찍혀 있었다.

LG 통신비만 연결해놓다가 실험을 위해서 KT와이브로를 넣어봤는데

KT모바일(와이브로) 가 먼저 찍히면서 2000P 가 지급된걸보니

올해 8개월차 까지는 와이브로를 유지해야 한다는 이야기인데..

현재 와이브로는 7개월 무료 체험중이며

7개월 이후에는 해지 예정이다

와이브로는 9월18일날 전화가 와서

9월24일 개통하여 받아 이용중이다.

7개월을 세어보자면 4월 24일까지 이용이 되는것이고

요금은 한달 늦으니 5월달까지 청구하게 될것

그리고 하나머니는 8월까지 혜택 받을 권리가 있다

이건 all in one 카드 이벤트이고

kt 리빙 카드 이벤트와는 연계가 안될수도 있다

더불어 핸드폰은 올해 2월달에 해지 예정

3월달까지는 요금이 청구가 되겠지?

3월달까지 나올 LG 와

5월달까지 나올 KT 와이브로

2014년 11월 3년 약정인 인터넷이

있으니 이건 20017년 11월까지 유지

이 카드로 2019년 3월쯤까지 5천원 할인

이니 인터넷인 계약만료후 1년 반쯤 더 유지해야 한다?

를 명제들을 잘 조합해보면

kt 와이브로는 all in one 카드로 5월까지 2000p ( 8월까지 가능한데 3개월 날림 )

KT 인터넷은 KT 1Q 리방카드로 결제하여 5000원 할인을 받으며

실적을 쌓기 위해서 첫달 결제 확인후에 정당원비를 매달 자동이체 시켜둔다

( 정당 원비는 연말정산 환급 가능 )

그럼 KDB산업은행 CHOICE 체크카드( 5%할인 ) 가 남는다 ㅡㅡ;;;

대형할인점 , 온라인 쇼핑 , 백화점, 병의원/약국 , 숙박 , 슈퍼 , 통신 , 레져 , 홈쇼핑 , 커피

통신비용은 더좋은 카드들로 교체할테니 온라인 쇼핑이랑 병원으로 바꿔야 할 듯

하나 머니 어플에서 보면 카드사용적립이 아닌 이벤트 적립이로 기재되어 있었다

날짜도 하루씩 뒤로 밀려 있는걸 보니 뭔가 이벤트 자체는 하나지만

적립 시스템이 1~3 / 4~12 개월의 이벤트가 적립 시스템이 틀린듯 하다
